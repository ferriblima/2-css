----- What is CSS? -----

CSS means Cascading Style Sheets

It can be used in three ways

  1. Inline CSS. Ex: <span style="color: red;"></span>
  2. Internal or Embedded CSS. That creates a <style type="text/css"></style> tag in the <head> (we can put it anywhere but is a practice to put in the head.
  3. External CSS. That creates a <style> tag in anothe file and it is imported in the html file.

There are three ways to link the internal and external css to one text

  1. Through the tag. p { second-style: green; }
  2. Through the class. .name-of-the-class { second-style: green; }
  3. Through the id. #name-of-the-id { second-style: green; }

When use id and when use class

  1. IDs should only be used a single time (only one instance of a specific id in the entire thing).
  2. Classes are not specific so can be used in all the pages.

What is a <div>? What is used for?

It's a Division, and is used to give styles to groups of code.

----- Positioning -----

There are three big types of positioning:

  1. float: right, left, ...;
  2. position: absolute, fixed, relative, static, ...;
  3. left:; right:; top:; bottom:;

Float

float: style to make it flows with the borders of the upper container.

float:left;

float:left;

float:right;

float:right;

clear: style that ignore the floats of other styles around or in the chosen side.

Obs.: A style inside other anulates the other.

Float right

Position

The position sets what is the reference of the block.

- position: relative; says that it will move relatively the position it was. It suffers interference of other blocks.

position: relative; left: 100px; top: 100px

Look that it moved 100px from the left and top border where it should be.

- position: absolute; says that it will move relatively the position of upper div. It ignores the other blocks. It doesn't flow with the page.

position: absolute; left: 1000px

- position: fixed; says that it will move relatively the position of upper div. It ignores the other blocks. It stays in the same position as the page moves.

position: fixed; top: 0; right: 0;

It must be used with top/bottom and right/left.
Chats and menus use this a lot.

How bring thing to the front?

The z-index: is used for this cases. The style with the greatest z-index stays in the front.
The z-index only work in positioned elements.

----- Margins -----

The difference between the positioning and the margin is that when we use the margin the blank space appears around all the block.

margin: 100px 70px 40px 10px; background-color:blue; width:300px; height: 100px; float: left;

margin: 50px 20px 50px 20px; background-color:blue; width:300px; height: 100px; float: left;

margin-top: 50px; margin-right: 20px; margin-bottom: 50px; margin-left: 20px; background-color:blue; width:300px; height: 100px; float: left;

----- Padding -----

In portuguese it means preenchimento and is similar to the margin, the difference is that it works inside other blocks.

  • margin = works outside the object.
  • padding = works inside the object.

padding: 50px 30px 15px 5px; background-color:blue; width:300px;

Obs.: The size of the object (width and height) sums with the paddings to result in the real size.

Obs.2: Naturaly, the browsers have a different default margins and paddings applyied to the pages. To remove it we can use a style in all of the <body>
    body { margin: 0; padding: 0; } and p { margin: 0; padding: 0; }

We can see the definitions for each browser looking for CSS Reset on google.
